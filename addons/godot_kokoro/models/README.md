# Kokoro model files

Model files are not included by default to keep the GitHub repository small and to make licensing/attribution explicit.

Expected files:

```text
addons/godot_kokoro/models/model.onnx
addons/godot_kokoro/models/voices.bin
addons/godot_kokoro/models/tokens.txt
```

Use `scripts/download_kokoro_model.sh` from the repository root, or download a compatible Kokoro model manually and place the files here.
